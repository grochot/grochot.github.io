Pandas obsługuje bardzo wiele typów danych, takich jak:

    >> Dane tabelaryczne z kolumnami ( np. SQL, Excel)

    >> Dane z szeregami czasowymi

    >> Dane macierzowe z etykietami kolumn i wierszy

    >> Inne zbiory danych obserwacyjnych

    Dane nie muszą być opisane, aby zostać umieszczone w strukturach danych

Podstawowe typy danych

DataFrame (2-wymiarowa)

Dwywymiarowa etykietownana struktura danych z kolumnami o potencjalnie różnym typie.

Podstawowe typy danych

Seria (1-wymiarowa)

Jednowymiarowa lista przechowująca różne typy danych.
Każda kolumna w DataFrame jest jednocześnie Serią.

Porównanie z Excelem

Pandas Excel
DataFrame Arkusz
Series Kolumna
Index Nagłówek wiersza
Wiersz Wiersz
NaN pusta komórka
